    XMLDom

    This class implements a small portion of the wc3 xml dom model. Generally useful for simple XML return objects. Note: stores namespaces in the attributes map for now. No node typing done at this time can parse into DOM trees the XML return objects from Google, Amazon and others. large parse trees will consume suprising amounts of memory
